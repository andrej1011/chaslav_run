
# Chaslav -  The Endless Runner
_A collaborative 2D endless runner built as a high-fidelity tribute to the Google Chrome Dino game._

## üë§ Project Story & Collaboration

This game was developed in 2019 as a collaborative project during our freshman year of high school. It was conceived as a challenging exercise to replicate a well-known, simple game mechanics of the endless runner using the Unity engine.

## ‚öôÔ∏è Technical Features

-   **Engine:** Unity 2019 (C#)
    
-   **Physics & Logic:** Custom-implemented game loop with dynamically increasing speed and obstacle frequency.
    
-   **Art Style:** Utilizes pixel art for graphics, successfully matching the minimalist aesthetic and feel of the source material.
    
-   **Core Systems:** Full custom implementation of crucial engine components: **object spawning**, **collision detection**, and **parallax background scrolling**.
